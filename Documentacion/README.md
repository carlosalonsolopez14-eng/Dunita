# Documentación de Dune Dominion

Esta carpeta contiene toda la documentación relacionada con el diseño, mecánicas y guías del juego **Dune Dominion**. Se ha reorganizado para separar claramente el contenido de la versión actual (Python/Pygame) del prototipo antiguo (C#).

## 📂 Estructura de la Documentación

### 1. `Guia_Jugador/`
Contiene los manuales y guías destinados a los jugadores finales.
- **MANUAL_DEL_JUGADOR.pdf**: Guía completa que explica el universo del juego, cómo empezar, estadísticas de edificios, criaturas, objetos y estrategias recomendadas.

### 2. `Diseno_Juego/`
Contiene documentos técnicos y de diseño sobre las mecánicas internas del juego.
- **Mecanicas y Gameplay/**: Carpeta con documentos detallados sobre el loop principal, sistema económico, fugas, reparaciones y eventos.
- **dune-creatures-api-data.json**: Base de datos en formato JSON con el lore y estadísticas extendidas de las criaturas.
- **DUNE_DOMINION_CATALOG.json**: Catálogo conceptual de instalaciones, defensas y objetos.
- **🦂 DUNE Criaturas API Explorer.pdf**: Documento exploratorio sobre la API de criaturas.

### 3. `Legacy_Prototipo/`
Contiene documentación obsoleta perteneciente a la versión anterior del juego (arquitectura web multi-agente en C#). Se mantiene únicamente como referencia histórica.
- **arquitectura-ai-zoo.html**: Diagrama interactivo de la antigua arquitectura de agentes.
- **zoo manager architecture.pdf**: Documento PDF sobre la arquitectura del prototipo.

### 4. `Contexto/`
Colección de documentos Word (`.docx`) con el lore y la historia del universo de Dune. Incluye información sobre facciones (Casa Atreides, Casa Harkonnen, Bene Gesserit, CHOAM), planetas (Arrakis, Caladan, Giedi Prime), personajes (Paul Atreides, Chani Kynes) y conceptos clave (Melange, Shai-Hulud).

### 5. `Imagenes-Estilo visual/`
Referencias visuales e imágenes de inspiración utilizadas para definir el estilo artístico del juego (fondos de menú, paletas de color, mapas de Arrakis).

### 6. `Musica/`
Pistas de audio originales del juego: música de menú, de partida, de eventos de peligro y de créditos.

### 7. `imagenes_edificios/`
Sprites originales de los edificios del juego antes de ser procesados a estilo pixel-art 8-bit.

---
*Nota: Si buscas información sobre cómo instalar o ejecutar el juego, consulta el `README.md` principal en la raíz del repositorio.*
