DUNITA

Dune Dominion Multi-Agent Simulator

Un simulador de gestión de zoológico/parque temático ambientado en el universo de Dune, construido con arquitectura multi-agente para una simulación realista y modular.

Visión General del Proyecto

Arquitectura Multi-Agente

El proyecto implementa una arquitectura jerárquica de agentes IA especializados:

  • Orquestador Principal: Coordina todo el sistema
  • Agente de Animales: Gestiona ciclo de vida de criaturas
  • Agente de Construcción: Maneja instalaciones y monetización
  • Agente de Mapas: Controla enclaves y territorio
  • Agente de Persistencia: Gestiona guardado y carga

Características

  • Simulación mensual de recursos y población
  • Sistema de enclaves con diferentes tipos
  • Gestión de criaturas con necesidades biológicas
  • Monetización basada en visitantes y atracciones
  • Persistencia de estado en JSON
  • Interfaz de consola interactiva

Cómo Ejecutar

Requisitos

  • .NET 8.0 SDK
  • Sistema operativo Linux/Windows/macOS

Ejecución

cd DuneDominion.Client
dotnet run

La aplicación se ejecuta en modo consola interactivo. Selecciona opciones del menú para gestionar tu parque.
